# Magicwall
## Usefull commands
### Project setup
```
npm install
```

#### Project run
Build application and run with electron
```
npm start
```

#### Build application
```
npm run buildonly
```

#### Run with electron
```
npm run eletron
```

#### Create package
```
npm run build
```

#### Create package without building application
```
npm run pack
```

## Details
Work on Windows

Work on linux desktop : xfce4, gnome, deepin
