# Magicwall

## 🔎 Details

* Search backgrounds by keyword
* Add on startup
* Change background when app is launched
* Choose resolution
* Add your API key (optional)

Work on Windows

Work on Mac

Work on linux desktop : xfce4, gnome, deepin, i3

## 🖥️ Usefull commands

### Project setup

```
npm install
```

#### Project run

Build application and run with electron
```
npm start
```

#### Build application

```
npm run buildonly
```

#### Run with electron

```
npm run eletron
```

#### Create package

```
npm run build
```

#### Create package without building application

```
npm run pack
```
